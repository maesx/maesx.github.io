# 辅助脚本

本目录包含项目的辅助脚本。

## 脚本说明

### quick_train.sh

交互式快速训练脚本，提供多种训练模式选择：

```bash
./scripts/quick_train.sh
```

选项：
1. 超快速验证 (~1小时) - 9% 数据集，10 轮
2. 快速训练 (~2小时) - 11.25% 数据集，20 轮
3. 标准快速 (~3小时) - 20% 数据集，30 轮
4. 完整训练 (~60小时) - 全部数据，50 轮

### train_and_infer.sh

训练 + 推理一体化流程脚本：

```bash
./scripts/train_and_infer.sh
```

功能：
- 自动检测是否存在训练好的模型
- 如无模型则先进行训练
- 训练完成后自动对验证集进行推理
